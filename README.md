# 🥗 Nutrition Tracking Bot

Персональный Telegram-бот для трекинга КБЖУ на сушке с AI-анализом.

## Быстрый старт

### 1. Получи токены

**Telegram Bot Token:**
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot`
3. Придумай имя и username для бота
4. Скопируй токен вида `123456:ABC-DEF...`

**Anthropic API Key:**
1. Зарегистрируйся на [console.anthropic.com](https://console.anthropic.com)
2. Перейди в API Keys → Create Key
3. Скопируй ключ вида `sk-ant-...`

**Твой Telegram User ID:**
1. Напиши боту [@userinfobot](https://t.me/userinfobot)
2. Он ответит твоим ID (число вида `123456789`)

---

### 2. Деплой на Railway (бесплатно)

1. Зарегистрируйся на [railway.app](https://railway.app) (через GitHub)
2. Нажми **New Project** → **Deploy from GitHub repo**
3. Залей эту папку в GitHub репозиторий (публичный или приватный)
4. В Railway выбери этот репо

**Добавь переменные окружения** (Variables tab):
```
BOT_TOKEN=твой_токен_от_botfather
ANTHROPIC_API_KEY=твой_ключ_anthropic
ALLOWED_USER_ID=твой_telegram_id
DB_PATH=/app/data/nutrition.db
```

5. Railway сам установит зависимости и запустит бота!

---

### 3. Локальный запуск (для теста)

```bash
# Установи зависимости
pip install -r requirements.txt

# Создай .env файл
cp .env.example .env
# Заполни свои ключи в .env

# Запусти
python bot.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| Просто текст | Добавить приём пищи |
| `/итог` | Сводка дня + AI-анализ |
| `/неделя` | Аналитика за 7 дней |
| `/план` | Твой план на сегодня |
| `/история` | Данные за 7 дней |

## Примеры сообщений боту

```
завтрак: овсянка 100г на воде, 3 варёных яйца, кофе с 50мл молока
обед: куриная грудка 200г, рис 150г, огурец
протеин 30г на воде
ужин: творог 200г 5%, орехи кешью 30г
```
